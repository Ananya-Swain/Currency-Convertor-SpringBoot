<h2>Currency Converter (Spring Boot + HTML/CSS/JS)</h2>

A simple and user-friendly Currency Converter web application built using Java Spring Boot for the backend and HTML, CSS, JavaScript for the frontend.
The application fetches live exchange rates from an external API and allows users to convert currencies instantly.
